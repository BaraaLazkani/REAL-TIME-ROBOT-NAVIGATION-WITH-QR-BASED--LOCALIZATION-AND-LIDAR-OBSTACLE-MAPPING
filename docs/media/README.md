## Usage

These videos demonstrate:
- ArUco marker detection and localization
- Kalman filter-based pose estimation
- Waypoint navigation with proportional control
- LIDAR-based obstacle detection and avoidance
- Real-time sensor fusion

For the technical implementation details, please refer to the main [documentation](../report.pdf).
